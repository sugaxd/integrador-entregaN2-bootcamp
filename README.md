# El Bordo - E-commerce de Instrumentos Musicales

## Descripción
Tienda online de instrumentos musicales desarrollada con React. Permite explorar productos, registrarse como usuario y gestionar el catálogo desde un panel de administración.

## Tecnologías utilizadas
- React 18
- React Router DOM v6
- MockAPI (REST API externa)
- CSS propio (Bootstrap solo en lo permitido)


## Instalación y ejecución

```bash
# Clonar el repositorio
git clone https://github.com/sugaxd/integrador-entregaN2-bootcamp

# Ingresar al directorio
cd el-bordo

# Instalar dependencias
npm install

# Ejecutar en modo desarrollo
npm start
```


## Páginas disponibles

| Ruta | Descripción |
|------|-------------|
| `/` | Home con productos |
| `/productos` | Listado general |
| `/productos/:id` | Detalle de producto |
| `/registro` | Formulario de registro |
| `/login` | Inicio de sesión |
| `/acerca-de` | Acerca de nosotros |
| `/admin/productos` | CRUD de productos |
| `/admin/usuarios` | CRUD de usuarios |
